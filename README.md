# EF- M6 Proyecto: "KanbanPro" - Sprint 1

Prototipo funcional de una aplicación Kanban con renderizado del lado del servidor y persistencia de datos en un archivo JSON.

## Funcionalidades

- Navegación entre Home, Registro, Login y Dashboard  
- Lectura de tareas desde `data.json`  
- Creación de nuevas tareas mediante formulario  
- Persistencia de datos usando el módulo `fs` de Node.js  

## Tecnologías

- Node.js  
- Express  
- Handlebars (hbs)  
- HTML / CSS  
- JSON  
- Nodemon
